# octopress-theme-ghost

An Octopress theme inspired by [Ghost](https://ghost.org)

## Installation

```
$ git submodule add https://github.com/xiaogaozi/octopress-theme-ghost.git .themes/ghost
$ rake install[ghost]
$ rake generate
```
